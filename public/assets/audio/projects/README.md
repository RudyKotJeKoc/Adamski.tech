# Audio dla sekcji Projekty (Projects)

## Sugerowane pliki audio dla każdego projektu:

### Format nazwy:
`[nazwa-projektu].mp3`

### Przykłady:

1. **machine-transfer-system.mp3**
   - Opis projektu systemu transferu maszyn
   - Wyzwania, podejście, rezultaty
   - Długość: 2-3 minuty

2. **smart-home-infrastructure.mp3**
   - Opis infrastruktury smart home
   - Technologie użyte: Home Assistant, ESPHome
   - Długość: 2-3 minuty

3. **daremon-radio-ets.mp3**
   - Opis projektu Daremon Radio ETS
   - Długość: 2-3 minuty

4. **mim-furnaces-proposal.mp3**
   - Opis propozycji modernizacji pieców MIM
   - Długość: 2-3 minuty

5. **youtube-channel.mp3**
   - Opis kanału YouTube i tworzenia contentu
   - Długość: 2-3 minuty

6. **industrial-automation-business.mp3**
   - Opis biznesu automatyzacji przemysłowej
   - Długość: 2-3 minuty

## Przykład integracji w content.json:

```json
{
  "projects": {
    "title": "Projekty",
    "items": [
      {
        "name": "Machine Transfer System",
        "tagline": "Retrofit systemu transportu międzymaszynowego",
        "audio": "/assets/audio/projects/machine-transfer-system.mp3",
        ...
      }
    ]
  }
}
```

## Wskazówki dotyczące nagrywania:

- Mów wyraźnie i w spokojnym tempie
- Zacznij od przedstawienia projektu
- Opisz wyzwanie, podejście i rezultaty
- Podkreśl kluczowe technologie i metryki
- Zakończ jednym zdaniem podsumowującym
