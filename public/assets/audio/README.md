# Struktura plików audio/video

Ten folder zawiera pliki audio i video dla różnych sekcji strony portfolio.

## Struktura katalogów:

```
/public/assets/audio/
├── about/          # Pliki audio dla sekcji "O mnie"
├── skills/         # Pliki audio dla sekcji "Umiejętności"
├── projects/       # Pliki audio dla sekcji "Projekty"
├── career/         # Pliki audio dla sekcji "Timeline kariery"
├── equipment/      # Pliki audio dla sekcji "Wyposażenie"
└── brand/          # Pliki audio dla sekcji "Filozofia Daremon"
```

## Obsługiwane formaty:

### Audio:
- `.mp3` (zalecane)
- `.wav`
- `.ogg`

### Video:
- `.mp4` (zalecane)
- `.webm`

## Konwencja nazewnictwa plików:

Używaj nazw opisowych w formie kebab-case:

**Przykłady:**
- `introduction.mp3` - ogólne wprowadzenie do sekcji
- `skills-overview.mp3` - przegląd umiejętności
- `project-automation.mp3` - opis projektu automatyzacji
- `career-2015-2020.mp3` - opis kariery w latach 2015-2020
- `equipment-fanuc.mp3` - opis sprzętu FANUC

## Zalecenia techniczne:

- **Format**: MP3, 128-192 kbps (balans jakość/rozmiar)
- **Długość**: 30-90 sekund dla wprowadzeń sekcji
- **Długość**: 2-5 minut dla szczegółowych opisów projektów
- **Rozmiar**: Staraj się utrzymać poniżej 5 MB na plik

## Jak dodać audio do sekcji:

Szczegółowe instrukcje znajdziesz w pliku głównym README w katalogu każdej sekcji.
